# 스프링 블로그 프로젝트 #
### 프로그램 셋팅 환경 ###
- JAVA 11v
- MySQL
- Spring boot STS
### 스프링 의존성 설정 ###
- Spring boot DevTools
- Lombok
- Spring Data JPA
- MySql Driver
- Spring Security
- Oauth2 client - 셋팅x
- Spring Web
### 템플릿 엔진 ###
- jsp

# DB 셋팅 #
- User ( 사용자 정보 )
- Board ( 게시판 정보 )
- Repley ( 답글 정보 )
